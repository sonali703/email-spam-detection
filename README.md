# 📧 Email Spam Detection (Flask + ML)

## 🚀 Features
- Detect Spam / Not Spam
- Machine Learning (Naive Bayes)
- Flask Web App
- Deployable on Render

## ▶️ Run Locally
```bash
pip install -r requirements.txt
python train.py   # run once
python app.py
```

## 🌐 Deploy
- Push to GitHub
- Connect to Render
- Start Command: `python app.py`
